# NutriScan – Public Health & Nutrition Chatbot

## Overview
NutriScan is a Public Health & Nutrition chatbot designed to provide users with simple and easy-to-understand nutrition guidance through a conversational interface.

This project was built using AI-powered platforms instead of manual frontend or backend development. The focus of this assignment is on problem understanding, AI integration, and practical usage of cloud-based AI tools.

---

## Problem Statement
Many people struggle to find reliable and easy-to-understand nutrition information. Most online sources are either too technical or inconsistent. NutriScan aims to solve this by providing a conversational chatbot that users can interact with naturally.

---

## Target Users
- General public  
- Students and working professionals  
- People interested in improving their food habits  

---

## Solution Approach
- A web-based interface hosts an embedded nutrition chatbot
- Users ask nutrition-related questions
- The chatbot responds with general health and nutrition guidance

The solution is intentionally simple and focused on usability.

---

## Tools and Platforms Used
- **Base44** – Used to build and host the website
- **Jotform AI Agent** – Used to create and embed the chatbot
- **Cloud-based Language Model** – Used internally by the AI agent to generate responses

No manual backend or model training code was written.
