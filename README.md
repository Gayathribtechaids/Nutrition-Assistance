**NutriScan – Public Health & Nutrition Chatbot**

Project Overview**
   NutriScan is a web-based Public Health & Nutrition chatbot designed to help users get simple and reliable nutrition guidance through a conversational interface. The chatbot allows users to ask questions related to food, healthy eating habits, and general wellness.
This project was developed as part of an assessment to demonstrate how cloud-based AI tools can be applied to solve real-world problems in a practical and user-friendly way.

**Problem Statement**

  Access to clear and trustworthy nutrition information is still a challenge for many people. Most users either rely on random online sources or avoid nutrition guidance altogether because it feels complex or confusing.

**Problem Being Addressed**

  Limited access to simple nutrition guidance

  Confusing or unreliable health information online

  Lack of interactive platforms for nutrition-related questions

**Target Users**

  General public

  Students and working professionals

  Individuals interested in improving their diet and lifestyle

**Reason for Choosing This Problem

   Nutrition plays a major role in long-term health. A simple chatbot can act as an easy starting point for users who want guidance without being overwhelmed.

**Solution Description**

 NutriScan provides a conversational chatbot embedded into a web application. Users can type nutrition-related questions and receive clear, easy-to-understand responses.

**The solution focuses on:**

  Simplicity

  Ease of access

  Clean and minimal design

**Use of AI and Cloud Services

  The chatbot uses a cloud-hosted language model to understand user questions and generate relevant responses.

**Tools and Platforms Used**

   Base44 – Used to build and host the web application

   Jotform AI Agent – Used to create and embed the chatbot

   Cloud-based LLM service – Used for natural language understanding and response generation

   The chatbot is embedded into the website using a script-based integration.

**System Flow (High-Level)**

  User enters a question on the website

  The embedded chatbot receives the input

  The cloud AI service processes the request

  A response is generated and shown to the user

**Data and Knowledge Scope**

   The chatbot provides general nutrition and wellness information. It is intended only for educational purposes and does not replace professional medical advice.

**Key Learnings**

  Applying AI tools to real-world problems

 Integrating cloud-based chatbots into web applications

 Using low-code platforms to speed up development

 Designing simple and usable systems

**Conclusion**

  NutriScan demonstrates how an AI-powered chatbot can be used to support public health awareness in a simple and accessible way. The project focuses on clarity, usability, and correct application of AI tools rather than unnecessary complexity.
